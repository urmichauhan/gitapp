Log IN
User needs a personal token to login GITAPP

You have to verify your username before login.
If user already logged in,username verification can be skipped.
In case of you don't have access token then go link "Create Token".

User View
Main or Home page.

Top right second Image button displays your current profile picture of github.
On clicking image button full view opens with your details like :
Username,Total stars,Following users,Your followers,public repositories and gists.
Also displays Bio and location.

News-Feed

On the middle of your screen moves slides of latest five Public Organization events.
Also it includes with some information like Author,date & time ,type etc..

Repositories

Displays all repositories of you have created.
On click any repository, user land to the repositories github page.

Gists

Displays all gists of you have created.
On click any gist, user land to the gist's github page.

Followers

List of your user who follows you.
on clicking any user, displays it's user view.

Following

List of your users whom you followed.
on clicking any user, displays it's user view.

Search
Search user

You can search a user from searchbox.
type username and press search icon displays list of possible most same username's user.
find out and you should land on that particular user view.

Footer
Contain Fields for users

About: displays information about Application.
